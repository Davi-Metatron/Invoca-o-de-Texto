# -------------------- # PORQUE ESSES PROGRMAS? 

    Esse é o primeiro de 20 programas basicos de um programador de uma lista de 5 niveis. Saber fazer um "Hello World"
em 5 linguagens para entender como funcionam. Tanto que o nome dessa pasta é "Invocação de Texto"
    Esse projeto consiste em transformar para mim programação em magia, assim como RGP, sendo esse para mim um "Feitiço".
Cujo o objetivo é entender como linguagens invocam texto.
